# First-Code
Apenas um teste simples para aprender e praticar.
É um script de verificação de lista com números inteiros específicos, algo simples apenas para praticar e aprimorar meus conhecimentos.

Para rodar o código é simples;
baixe ele no seu computador, após isso, vá no terminal e digite: "python3 Code_Chalenge.py".
Após isso pedirá um número que você deseja verificar dentro da lista, insira o número e o código retornará uma mensagem dizendo se existe ou não este número.
